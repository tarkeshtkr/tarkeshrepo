Data Cleaning Scripts

1. Remove Duplicate Rows

DELETE t1 FROM your_table_name t1 JOIN your_table_name t2 ON t1.column_name = t2.column_name AND t1.id > t2.id; Copy Script

2. Remove Rows with NULL Values

DELETE FROM your_table_name WHERE column_name IS NULL; Copy Script

3. Replace NULL with Default Value

UPDATE your_table_name SET column_name = 'default_value' WHERE column_name IS NULL; Copy Script

4. Trim Whitespace from String Columns

UPDATE your_table_name SET column_name = TRIM(column_name) WHERE column_name IS NOT NULL; Copy Script

5. Standardize Date Formats

UPDATE your_table_name SET date_column = STR_TO_DATE(date_column, '%m/%d/%Y') WHERE STR_TO_DATE(date_column, '%m/%d/%Y') IS NOT NULL; Copy Script

6. Remove Non-Alphanumeric Characters

UPDATE your_table_name SET column_name = REGEXP_REPLACE(column_name, '[^a-zA-Z0-9 ]', '') WHERE column_name IS NOT NULL; Copy Script
